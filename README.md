# Sales Performance Dashboard

This repository contains a Python notebook analysis of the Superstore sales dataset.

## Files

- `dataset/superstore_sales.csv` – original Superstore sales dataset.
- `notebooks/sales_analysis.ipynb` – analysis notebook with charts and KPI calculations.
- `images/` – folder for storing exported charts or visuals used in documentation.

## How to run

1. Install the required Python packages (pandas, matplotlib, seaborn).
2. Open `notebooks/sales_analysis.ipynb` in Jupyter or VS Code.
3. Run the notebook cells in order.

## Notes

This project is intended to be used as the basis for creating a Power BI dashboard from the cleaned/exported dataset.
